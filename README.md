<h1 align="center">unity</h1>

### Information:
unity - open source file manager written on C++ for unix-like systems

### Motivation:
Experience

### Install:
1. Clone repository 
  > git clone https://github.com/vladikvt/unity
2. Enter in unity folder 
  > cd unity
3. Compile project 
  > make
4. Run unity
  > ./unity
